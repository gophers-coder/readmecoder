# README

> 教程项目规范


## 1。命名

```text
主题 + coder

1. 没有大小写
2. 没有下划线
3. coder 结尾

```

比如：

`shellcoder`

## 3。项目需要包含

- travis.yaml : travis ci 持续集成
- Dockerfile : 构建镜像 + DaoCloud
- .gitignore: 项目忽略上传的文件
- package controller: 版本控制 go mod
- Makefile: 项目编译程序
- 在线访问网址：使用 github pages
